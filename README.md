react + golang 前后端分离的博客

### 功能

- [x] 文章展示
- [x] 标签
- [x] 归档
- [x] 其他页面
- [x] 单用户登录
- [x] 后台管理
- [x] Hexo兼容
- [x] RSS

### 使用

1. 安装postgresql，并新建一个数据库
2. 修改`config/config.yaml`配置文件
3. 在NGINX或caddy上配置静态文件目录和反代

#### Markdown格式

Markdown

#### hexo文章的导入

在编辑器内粘贴hexo的`_posts`文件内容就能自动导入，在设置里可以一键上传多个文件
